API's

API staat voor application programming interface. Door middel van Javascript kunnen gegevens van bronnen van bijvoorbeeld het internet gehaald worden en live weergegeven worden op je website. Ik heb gebruik gemaakt van een OpenWheaterAPI, deze is rechtsboven op het beeldscherm te vinden, hier wordt live het weer van Nijmegen weergegeven.

Daarnaast heb ik ook nog een API op mijn website staan die de live gegevens omtrent het COVID-19 virus weergeven, deze is te vinden onder het kopje Statistieken
